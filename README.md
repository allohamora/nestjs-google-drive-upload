# Nestjs Google Drive Upload

a nestjs google drive upload app

## Installation

```bash
$ npm i
```

## Running the app

```bash
# development
# create and fill .env from .env.example
$ docker compose up -d
$ npm run start:dev
```

## Migrations

```bash
# generate a migration
$ npm run migration:generate --name=test

# run a migration
$ npm run migration:run

# revert a migration
$ npm run migration:revert
```
